Documentation

BulmaJS is an unofficial Javascript extension for the Bulma CSS framework.

{% include notice.html %}

Using ES6 versions

This requires a build process to be in-place to transpile the ES6 code to ES5.

The easiest way to get started with BulmaJS is to load everything.

{% highlight javascript %} {%- include code/import-src.js -%} {% endhighlight %}

For production I would recommend only including the plugins you need. So, if you need to use Notification and Navbar.

{% highlight javascript %} {%- include code/import-singular.js -%} {% endhighlight %}

Using ES5 versions

Useful if you do not have a build process in place.

If you would prefer to include the JS files directly in your HTML, without any transpiling you can still do this through the dist folder. This method provides limited flexability and so is not the recommended way for production.

If you would like to include a single plugin you can include that plugin using the below syntax. This should be placed in the closing body of your page, so that the DOM has loaded ready for the plugin to search for any DOM apis.

This method is not recommended if you are using more than one plugin, as each independent file has the core packaged with it. It is recommended to instead use the ES6 method to build your own collection.
{% highlight html %} {%- include code/import-es5-singular.html -%} {% endhighlight %}

Finally, to include all of the BulmaJS plugins, you include the bulma.js file inside the dist folder.

{% highlight html %} {%- include code/import-es5-all.html -%} {% endhighlight %}
