Accordion

since 0.3.0
DOM API Yes
JS API Yes
{% include notice.html %}
This is a BulmaJS ES6 plugin to integrate the JS functionality needed to use Wikiki's Accordion Bulma extension. BulmaJS does not come with the CSS styling, this will need to be downloaded separately.
