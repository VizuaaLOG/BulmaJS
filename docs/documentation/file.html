File name updating

The Bulma file form element contains a place for the file name to be placed. This plugin adds the functionality to update the file-name element.

If no file is selected, i.e. the open dialog is cancelled, the file name element will be cleared. If a single file is selected then the name of the file will be inserted. If multiple files are selected then the the text will be updated with x files.
