Methods

Method name Parameters Description
show() None This will show the message (remove the is-hidden class.
hide() None This will hide the message (add the is-hidden class.

Javascript options

Option name Required Default Type Description
body Yes '' String This is the body of the message.
parent No document.body HTMLElement This is the element you would like the message to be appended to.
color No '' String This is the color modifer to add to the message. For example for is-info you would set color to info.
size No '' String This is the size modifer to add to the message. For example for is-large would make the message the larger variant.
dismissInterval No null Integer This is the time in milliseconds the message is visible.
isDismissable No false Boolean Would you like the message to be dismissable, we will also add the delete button to the rendered HTML.
destroyOnDismiss No true Boolean Would you like the message to be destroyed when dismissed? This will remove the HTML and also unregister events.
element No true HTMLElement This is a reference to a .message element. Generally you do not need to include this value, it is primarily used for the automatic DOM parsing.
closeButton No <button class="delete"></button> HTMLElement This is the element you would like to be used when dismissing the message.

DOM options

Option name Required Default Type Description
data-dismiss-interval No null Integer The amount of time in milliseconds until the message is automatically dismissed.
